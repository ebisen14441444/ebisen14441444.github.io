HTMLを書いてみよう！

Webページの「骨組み」を作ってみよう

CodePen を開く ↗
無料ですが、会員登録しないと保存はされません。登録したくない人は、うっかりブラウザバックやリロードをしてしまわないように注意しましょう。可能なら登録した方が後の部分で共有できて便利です。

HTMLとは

  • Webページの構造を作るための言語です。
  • 「タグ」という <> で囲んだ命令を使って書きます。
  • 開始タグ <h1> と終了タグ </h1> でテキストを挟みます。

色々なタグ

適当にCodePenにコピペして、どういう変化をするのか弄ってみましょう

  • コメントアウト : <!-- うおおお -->

    <!-- と --> で囲んだ部分はブラウザに表示されません。「ここ何のコードだっけ？」ってメモを残したいときに便利です。

    使い方
    <!-- これはコメントです。ブラウザには表示されません -->
<p>これは表示されます</p>
  • <h1> ～ <h6> : 見出しタグ

    h1 が一番大きく、数字が増えるほど小さくなります。記事のタイトルや章タイトルみたいなイメージで使いましょう。

    使い方
    <h1>一番大きい見出し</h1>
<h2>二番目に大きい見出し</h2>
<h3>三番目に大きい見出し</h3>
  • <div> : グループ分けタグ

    「ここからここまでひとまとめ！」って箱を作るタグです。見た目は何も変わりませんが、あとでCSSでスタイルをつけるときにとても役立ちます。

    使い方
    <div>
  <p>この段落はdivの中にある</p>
  <p>こっちも同じグループ</p>
</div>
  • <ul> と <li> : リスト（箇条書き）

    箇条書きリストを作るタグです。<ul> がリスト全体の箱で、<li> が各項目。<ul> を <ol> に変えると番号付きリストになります。

    使い方
    <!-- 順番なしリスト -->
<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>

<!-- 順番ありリスト -->
<ol> 好きな果物ランキング
  <li>りんご</li>
  <li>みかん</li>
</ol>
  • <p> : 段落タグ

    文章のかたまりを書くときに使うタグです。段落ごとに <p> で囲む習慣をつけておくと、あとでCSSで整えるのが楽になります。

    使い方
    <p>ここに文章を書きます。</p>
<p>別の段落はまた別のpタグで囲みます。</p>
  • <br> : 改行タグ

    文章を改行するタグです。</br> みたいな終了タグは不要で、<br> だけでOKです。

    使い方
    <p>一行目の文章<br>
ここから二行目になります</p>
  • <a> : リンクタグ

    リンクを貼るタグです。href="URL" でリンク先を指定します。target="_blank" をつけると別タブで開けます。

    使い方
    <!-- 普通のリンク -->
<a href="https://example.com">ここをクリック</a>

<!-- 別タブで開くリンク -->
<a href="https://example.com" target="_blank">別タブで開く</a>
